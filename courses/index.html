Course

Applied Financial Economics

Description

  • We cover how theoretical models and the associated empirical analysis can be applied in practice
  • We first review classical stock market analysis techniques: fundamental analysis and technical analysis.
  • We investigate various trading strategies adopted by market participants.
  • We continue to study how modern time series models can help to analyze and predict future stock price.
  • Then we examine several market microstructure models. trading models. to understand trading behavior such as bid-ask spread, liquidity, trading volume, and trading behaviors.

Overview

  • Total time for the course is about 240 minutes.
  • There are eight topics and each topic is about 30 minutes.
  • Slides for this course is downloadable at Gumroad (name-your-own-price).

Topics

  • Overview
  • Price
    • Fuandamental Analysis
    • Technical Analysis
    • Factor model
    • Event study
  • Trading
    • Bid Ask Spread
    • Inventory Model
    • Sequential Trade
    • Behavioral Model

Short Course on LaTeX

Description

  • We cover how to write a basic LaTeX document
  • We learn how to do basic programming techniques in LaTeX
  • We show how slides can be created using beamer package
  • We examine how to draw diagram using TikZ package

Overview

  • Total time for the course is about 2 hours
  • It is divided into two parts: basic and advanced courses.
  • Each part is about one hour.
  • Slides and tex files for this course are downloadable at Gumroad (name-your-own-price).

Topics

  • Basic
    • hello world, front matter, sectioning, appendix, reference
    • mathematics, formal statements
    • tabular, table
    • graphics, figure
  • Advanced
    • Programming: newcommand, newenvironment, conditional if, for loop
    • Beamer
    • TikZ: line, node, curve, circle, rectangle, ellipse, grid

Financial Analysis using Excel VBA

Overview

  • Total time for this course is about 3 hours
  • It is divided into three parts: Excel Spreadsheet, VBA basics, and VBA programming
  • Slides and Excel files for this course are downloadable at Gumroad (name-your-own-price).

Description

  • We cover how to do basic computation work using Excel
  • We learn how to do technical analysis using Excel
  • We learn how to do programming in Excel VBA
  • We cover how to use Excel VBA to streamline repetitive work in Excel
  • We learn how to use Excel VBA to enhance capability of Excel

Topics

  • Excel
  • Basic VBA
    • Variable, Function, and Subroutine
    • Objects: Range and Application
  • Advancded VBA
    • Conditional Statements: If, If-else, If-else-if, Case When
    • Repetition: While Loop, For loop

Description

  • We cover how to do basic computation work using R

  • We learn how to do programming in R
